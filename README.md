# cyclistic-bike-share-analysis
End-to-end data analysis of Cyclistic bike-share data using SQL, Python, and Tableau to uncover rider behavior patterns and deliver data-driven marketing recommendations for increasing annual memberships.

Cyclistic Bike Share Analysis

From Rider Behavior Analysis to Membership Growth Strategy
📌 Project Overview
This project analyzes one year of Cyclistic bike-share trip data to understand the behavioral differences between annual members and casual riders.
The primary business objective is to identify opportunities to convert casual riders into annual subscribers through data-driven marketing strategies.

This case study follows the complete Google Data Analytics workflow:

The business objective is to understand how different rider groups use Cyclistic bikes and provide actionable recommendations for increasing annual memberships.
